Todo App

A simple and user-friendly Todo application built to help users manage tasks efficiently while also providing a built-in timer with alarm functionality.

Features

📝 Todo List Page

- Displays an "Empty Todo" message by default when there are no tasks available.
- Users can add new tasks to the list.
- Tasks can be managed and removed as needed.
- The default message automatically disappears when tasks are added and reappears when all tasks are deleted.

⏱️ Timer Page

Accessible through the navigation menu.

Timer Features

- Set a countdown timer for any desired duration.
- Includes an optional alarm system.
- Clicking the Alarm Button activates the alarm and displays a red indicator light, showing that the alarm is enabled.
- When the countdown reaches zero:
  - If the alarm is enabled, a sound notification plays.
  - If the alarm is disabled, the timer simply ends without any sound.

How It Works

1. Open the Todo List page to manage your tasks.
2. Add tasks to keep track of your activities.
3. Navigate to the Timer page using the navigation menu.
4. Set a timer duration.
5. Enable the alarm by clicking the alarm button.
6. Confirm the red light indicator appears.
7. Start the timer.
8. When the timer finishes, the alarm sound will play if the alarm is active.
